Existen dos tipos Estatica y Dinamica

Elemento Form define un formulario que se utiliza para recopilar informacion que el usurio escribe

Elemento input(campos) es el mas importantes en un formulario, puede ser mostrado de diferentes formas dependiendo del atributo type.

Resumen

Los formularios son basicamente una forma de guardar informacion en el servidor, ya se para iniciar secion, o para hacer algun tipo de tramite, o reservacion, incluso para agendar una cita, por eso es importante implementarlos cuando se es necesario.
