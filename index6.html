Footer:
El footer o pie de página es una sección olvidada en muchos proyectos web, pero si es bien diseñado, puede ser muy útil y efectivo. El footer es un espacio genérico que suele tener el mismo diseño.
Planificar cada uno de los elementos que se van a incluir en el footer es parte del proyecto de diseño de un proyecto web. Así se establece los elementos que se van a mostrar Textos legales son elementos obligatorios en el footer de toda web y no hay lugar para interpretaciones. La privacidad de los usuarios que visitan la web es un tema muy serio y además es obligatorio. Una web tiene dos alternativas, tener avisos legales, ser infractor de la ley.
Los textos legales obligatorios La política de cookies es también uno de los textos legales obligatorios en todo sitio web. La política de cookies debe aparecer en forma de barra o mensaje Las redes sociales son otro de los elementos que es recomendable colocar en el footer y no en el header. Permite una vía de contacto directa entre la marca y las vistas de la web que se pueden convertir en seguidores. Se pueden añadir los botones de cada plataforma o utilizar widgets para mostrar las ultimas publicaciones.
Copyright: otra información que es recomendable colocar en el footer, es la referencia al copyright, ósea los créditos del diseñador, los autores del material gráfico como fotos ilustraciones o videos, instituciones que colaboran con el proyecto, si es el caso. Los datos de contacto son una información muy útil para que las visitas se pongan en contacto con la empresa o negocio. Se pueden incluir datos como teléfono, email, dirección, incluso se puede incrustar un mapa de google maps.

Header : aprenda a identificar los elementos
El haedaer es una de las partes mas importantes de en una pagina web pues es el elemento que los usuarios visualizan en primer lugar cuando llegan al proyecto Por lo tanto, la cabecera de la web debe ser ordenada y limpia para que pueda ser visualizada. Debemos tener en cuenta que el header es un espacio generico que tendra el mismo diseño en todas las paginas de un sitio web. Lo primero que debes hacer es un listado con los elementos que vas a incluir en el headerde tu web de forma obligatoria, ya sea por motivis de branding, navegacion u otros. Los elementos que pueden formar parte del header, van a depender de los objetivos y del tipo de web
Logotipo, menú, buscador, datos de contacto, etc. En una web no puede faltar el logotipo, sirve como guía visual para el usuario y permite reforzar la imagen de la marca. El logotipo en una web suele enlazar con la página de inicio. Es decir, permite que el usuario pueda volver al principio en todo momento Normalmente se encuentra esquina superior izquierda.
El menú o nav permite al usuario la navegación entre las distintas secciones. Puede ser también agregado al header. Se puede colocar un solo menú o dos, siempre en función del proyecto y de su estructura, si el proyecto tiene muchas secciones, posiblemente se necesite un menú doble, pero hay que Peor si el proyecto tiene una gran estructura o muchas categorías, en ese caso se necesitará un doble navegador o menú. Otro elemento que puede estar en el header de una web es el buscador, es por cuestiones de usabilidad, ya que tu conoces muy bien tu web, pero para un usuario que entra por primera vez es totalmente desconocida.
Los datos de contacto Las redes sociales son un canal complementario para comunicar el mensaje de la marca o negocio, pero no el principal, que siempre debe ser la web. El objetivo de las redes sociales es amplificar el mensaje para convertirlo en tráfico hacia la web, no al contrario. No tiene sentido que, cuando se ha conseguido que el usuario llegue a la web, lo primero que hagas sea sacarlo de ella para llevarlo a las redes sociales. Si la web es una tienda online, se debe incluir en el header los elementos referentes al proceso de compra: carrito, cuenta, pedido, lista de deseos, inicio de sesion, etc. Otro tipo de elemento que se puede incluir en la cabecera de tu web son botones destacados para enfatizar un servicio, llevar al usuario a una landing o animarle a que haga una acción especifica. Barras fijas son muy útiles para mostrar en el header de tus web anuncios, descuentos, cupones, llamadas, a las acción o información adicional. Si las usas, ten cuidado de no cargarlas de información para no restar protagonismo al resto de elementos.
Si el proyecto web es un sitio multi lenguaje un elemento obligatorio en el header es un selector de idioma. El formulario de suscripción puede ser incluido en el header, aunque no es recomendable por el gran espacio que ocupa. Además, los formularios de suscripción se pueden colocar en otras partes de la web como el footer.
